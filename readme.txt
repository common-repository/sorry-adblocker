== Description ==

This plugin will detect users using AdBlock software Crystal and other AdBlock tools.

AdBlocker can't see lasted post. Site owner can edit message title and body.

= Theme support =

For work this plugin, needs two HTML codes.
Post container elements (most themes use article tag) needs ID attribute include post ID. Like [id="post-123"].This number is the post ID.

And post content container needs class attribute "entry-content".

If don't work this plugin, check and edit theme.

= Supporting themes =

- Twenty Fifteen
- Twenty Fourteen
- Twenty Twelve
- Zerif Lite
- Customizr
- and other almost themes

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
